### SessionDetails
The user session details.


|Name|Description|Schema|
|---|---|---|
|**sessionId**  <br>*required*|The session ID.|string|
|**username**  <br>*required*|The name of a user associated with the session.|string|

**Example**
```
{
  "sessionId" : "i3h2bp4TjPVuOyvXulbW",
  "username" : "someName"
}
```
