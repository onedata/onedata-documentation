### UserDetails
The user configuration details.


|Name|Description|Schema|
|---|---|---|
|**userId**  <br>*required*|The user ID.|string|
|**userRole**  <br>*required*|The user role, one of `admin` or `regular`.|enum (admin, regular)|

**Example**
```
{
  "userId" : "i3h2bp4TjPVuOyvXulbW",
  "userRole" : "admin"
}
```
