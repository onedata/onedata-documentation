### FileAttribute
File attribute.


|Name|Description|Schema|
|---|---|---|
|**name**  <br>*optional*|Name of the attribute (e.g. 'mode')|string|
|**value**  <br>*optional*|Value of the file attribute. Can be|string|

**Example**
```
{
  "name" : "mode",
  "value" : "0444"
}
```
