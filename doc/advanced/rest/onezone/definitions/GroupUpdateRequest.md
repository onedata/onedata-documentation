### GroupUpdateRequest
Group update request.


|Name|Description|Schema|
|---|---|---|
|**name**  <br>*optional*|The name of the space.|string|
|**type**  <br>*optional*|The type of the group.|enum (organization, unit, team, role)|

**Example**
```
{
  "type" : "unit"
}
```
