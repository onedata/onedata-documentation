### UserUpdateRequest
User update request message.


|Name|Description|Schema|
|---|---|---|
|**alias**  <br>*optional*|User alias|string|
|**name**  <br>*optional*|The user friendly name of storage resource.|string|

**Example**
```
{
  "name" : "Rudolf Lingens",
  "alias" : "John Doe"
}
```
